Hypervisor

Software that lets one computer run multiple virtual systems.

  • Type 1 (Bare-metal): Hypervisor runs directly on hardware
  • Type 2 (Hosted): Hypervisor runs on top of an OS

Cloud Storage Types

  1. Cloud Storage Apps: Store files online (OneDrive, Google Drive, Dropbox)
  2. File Synchronization: Keep files updated across devices automatically
  3. Content Delivery Networks (CDN): Servers worldwide that deliver content fast

Cloud Deployment Models

  • Public Cloud: Resources shared online, cheap and common
  • Private Cloud: Company manages their own cloud internally
  • Hybrid Cloud: Mix of public and private resources
  • Community Cloud: Shared between organizations with similar needs

Cloud Access Security Broker (CASB)

Helps secure cloud use (Single sign-on, Access controls, Malware scanning, Activity monitoring).

Examples: Forcepoint, Blue Coat, Skyhigh Security, CloudLock
