Types of Cloud Computing

1. Public Cloud

Shared platform accessible to general public via internet. Pay-as-per-use.

  • Examples: AWS EC2, Microsoft Azure, Google Cloud.
  • Pros: Low cost, location independent, scalable, quick setup.
  • Cons: Lower security (shared resources), less customizable.

2. Private Cloud

Internal/Corporate cloud. Resources dedicated to one organization.

  • Examples: HP Data Centers, Ubuntu, Microsoft.
  • Pros: More control, high security & privacy, better performance.
  • Cons: High cost, skilled staff required, limited scalability.

3. Hybrid Cloud

Combination of Public and Private. Critical data stays private; non-critical goes public.

  • Pros: Flexible, cost effective, risk management.
  • Cons: Complex networking, infrastructure compatibility issues.

4. Community Cloud

Shared by several organizations with similar concerns/goals.

  • Pros: Cost sharing, flexible.
  • Cons: Diffused responsibility, fixed storage shared among members.
