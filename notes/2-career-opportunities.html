What is Cybersecurity?

Protecting computers, networks, and data from hackers, damage, or unauthorized access.

Red Team vs Blue Team

Red Team (Offense)
  • Try to hack systems before real attackers do
  • Simulate real attacks to find weak spots
Blue Team (Defense)
  • Protect systems before hackers find gaps
  • Monitor traffic, use firewalls, and stop unauthorized access

Security Operations Center (SOC)

Monitors and responds to security events. Helps prevent cyber attacks and keeps the system secure.

Three Pillars of Cybersecurity

  • People: Users and staff are key
  • Processes: Rules and steps to handle security
  • Technology: Tools like firewalls, antivirus, monitoring systems

All three need to work together to stay safe.

Vulnerability Scanning vs Penetration Testing

Vulnerability Scanning: Check systems for security weaknesses using tools. Automated tools find problems, but humans need to verify them.

Penetration Testing: A test hack to see if vulnerabilities are real. Also checks how IT staff respond and follow security rules.

Penetration Testing Life Cycle
  1. Recon: Gather info about the target without touching it
  2. Scanning/Enumeration: Actively probe defenses
  3. Gaining Access: Exploit a vulnerability to get in
  4. Maintaining Access: Keep control of the system (backdoors, etc.)
  5. Covering Tracks: Delete logs and history to hide actions
  6. Reporting: Document findings and suggest fixes
